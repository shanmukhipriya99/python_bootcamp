data types:
    1. Integer-  int
    2. Floating Point-  float
    3. Strings- str
    4. Lists-  list  (ordered sequence of objects) [10, 'hello', 200.4]
    5. Dictionaries- dict  (unordered key:value pairs) {"name": "Felix"}
    6. Tuples- tup  (ordered immutable sequence of objects) (10, 'hello', 200.4)
    7. Sets-  set  (unordered collection of unique objects) {'a', 'b'}
    8. Booleans- bool  (True/False)

=> type(variableName) returns the data type of the variable
=> index            0  1  2  3  4
   character        h  e  l  l  o
   reverse index    0 -4 -3 -2 -1
=> slicing syntax: [start:stop:step]
                    start: index for the slice start
                    stop: index you will go up to (but not include)
                    step: size of the jump
